# GregTech: Golden Core

## Setup

To add Golden Core to your mod, add this to your `build.gradle`, replacing `v0.3.0-alpha` with the latest version

```groovy
dependencies {
    modImplementation 'com.github.GT-Golden:GT-Golden-Core:v0.3.0-alpha' {
        transitive false
    }
}
```
